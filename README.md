# SUरुप्प - Authentic Nepali Tea Ordering System

A modern web application for ordering authentic Nepali tea directly from your table.

---

## 🎯 Features

- ☕ Digital menu with tea carousel navigation
- 🎵 Music request system  
- ⭐ Customer reviews
- 📱 Fully responsive design
- 💾 LocalStorage for data persistence
- 🔐 Admin portal with login

---

## 📁 Project Files

```
├── index.html          # Redirect page
├── home.html           # Home page
├── menu.html           # Menu & ordering
├── about.html          # About page
├── music.html          # Music requests
├── review.html         # Customer reviews
├── admin.html          # Admin dashboard
├── login.html          # Admin login
├── index.css           # Main styles
├── fab.css             # FAB styles
├── modal.css           # Modal styles
├── music.css           # Music styles

```

---

## 🚀 Pages

- **Home** - Hero section with how it works
- **Menu** - 6-item carousel with order form
- **About** - Company information
- **Music** - Request songs for playlist
- **Reviews** - Share and read reviews
- **Admin** - Staff dashboard

---

## 🛠 Technologies

- HTML5, CSS3, JavaScript
- Font Awesome icons
- Google Fonts (Playfair Display, Poppins)
- LocalStorage for data

---

## 🚀 Getting Started

1. Open `index.html` in a web browser
2. Navigate through pages using the menu navigation
3. Add items to order from the menu page
4. Request music and write reviews
5. All data is saved automatically in your browser

---

## 📝 Project Notes

**This is a group project built with only HTML, CSS, and JavaScript—no backend infrastructure.** All data is stored locally in the browser using LocalStorage, making it perfect for learning web development fundamentals and frontend design.
